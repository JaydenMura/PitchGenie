**PitchGenie** is an AI-powered outreach automation platform built to help businesses and professionals generate high-quality sales content in seconds.

---

## What It Does
- **Cold Email Generator** – Create personalized outreach emails with A/B variants.
- **Follow-Up Generator** – Write professional follow-ups to boost response rates.
- **Proposal Builder** – Generate complete proposals with clean formatting and PDF export.
- **Cold Call Script Generator** – Build engaging call scripts in seconds.
- **Objection Handler** – Overcome common objections with persuasive responses.

---

## Tech Highlights
- **Next.js (App Router)** + **React** for a modern, responsive frontend.
- **TailwindCSS** for fast, clean UI design.
- **OpenAI GPT-4** for advanced AI content generation.
- **Supabase** for data handling & user management.
- **Stripe** integration for subscription and billing features.
- Written in **TypeScript** for type safety and scalability.

The Template Style option allows users to choose how the final proposal is structured and formatted. It controls the presentation style, not just the wording.

Examples of Template Styles:

Standard – A formal, well-structured layout suitable for general business use.

Creative – An engaging, expressive, and visually distinct layout designed for dynamic or visually-oriented audiences.

Minimalist – A clean, concise layout focused only on essential information.

This setting helps tailor proposals to match the target audience’s needs and context, whether it's for a traditional business proposal or a pitch in a more creative industry.

This tool generates highly personalized cold emails tailored to your outreach goal. In Advanced Mode, users can input:

Call to Action (CTA) to drive engagement

Reference Point, like a shared event or contact, to build trust

Template Style to change the formatting (Standard, Minimalist, Creative)

Every email is generated with A/B versions to allow quick testing of messaging performance. This tool is ideal for freelancers, SDRs, founders, and anyone doing outbound email campaigns.


<img width="801" height="897" alt="Follow Up - Adv" src="https://github.com/user-attachments/assets/a458e9ba-d5da-48cf-81ca-0e1130565478" />

This tool helps craft polished follow-up emails that re-engage prospects or clients after an initial interaction. In Advanced Mode, users can input:

Previous Interaction Summary to add clarity and relevance

Call to Action (CTA) to prompt the next step

Template Style (Standard, Minimalist, Creative) for layout customization

Each follow-up is generated with two unique versions (A/B), giving users flexible options to test engagement. Perfect for closing deals, reminding prospects, or reviving cold leads.


<img width="777" height="870" alt="Cold Call Script - Adv" src="https://github.com/user-attachments/assets/34a410d0-95ca-43e5-a449-e55abc00fb08" />

This tool generates customized cold call scripts designed to grab attention, build rapport, and communicate value—quickly and effectively. In Advanced Mode, users can specify:

Context for the Call (e.g., prospect’s current challenges or pain points)

Template Style (Standard, Minimalist, Creative)

Each call script is generated in two unique A/B versions, making it easy to test different approaches and improve conversions during outbound sales efforts.


<img width="803" height="843" alt="Objection Handler - Adv" src="https://github.com/user-attachments/assets/c085c877-b462-48ba-b33c-63bd86adb298" />

The Objection Handler Generator helps sales professionals, founders, and SDRs overcome common objections during outreach or negotiation. Whether it’s about pricing, timing, or trust, this tool creates tailored responses to help close deals and build credibility.

With Advanced Mode enabled, users can customize:

Context or Additional Notes – Extra information that gives nuance to the situation or client.

Like all tools in the PitchGenie suite, it includes:

Template Style Selector – Choose how the response is structured:

Standard, Creative, or Minimalist

Tone Selector – Pick from tones like Professional, Friendly, Bold, etc.

Language Options – Write in English or other supported languages.

A/B Version Output – Generates two distinct variations (A and B) to test tone, phrasing, or strategy.


<img width="449" height="929" alt="A and B Generation" src="https://github.com/user-attachments/assets/9f98c979-d86c-4468-bb8c-8e2ea3291790" />

Each tool generates two distinct outputs (Version A and Version B) by default, giving users alternative messaging options.


<img width="753" height="468" alt="Language Options" src="https://github.com/user-attachments/assets/cce72783-f634-432a-8c9c-0a647169c17d" />

Users can select their preferred output language from a wide range of supported options, including English, Spanish, French, German, Japanese, Arabic, and many others. This ensures the generated content is globally accessible and adaptable to different markets and audiences. Changing the language modifies the output natively, no translation plugin or workaround is needed.

Use case: A user targeting a Korean or Spanish-speaking client can instantly generate outreach content tailored to that language and tone.


<img width="521" height="144" alt="Writing Style" src="https://github.com/user-attachments/assets/0164adc5-b500-41d7-bd8c-67df6e9edf25" />

This dropdown allows users to choose the tone and voice of the generated content to better align with their target audience or brand identity. The options include:

Professional – Formal, polished, and ideal for corporate communications

Friendly – Approachable and conversational without being overly casual

Persuasive – Bold, compelling copy designed to convert and sell

Casual – Relaxed and informal, suited for laid-back or younger audiences

This option applies across all tools (e.g., Proposal, Cold Email, Objection Handler) and enhances adaptability for different use cases.
